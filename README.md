# KYK-Yurt-Yonetimi-Sistem-Simulasyonu
Veritabanı Yönetimi Dersi için hazırlanmış bir projedir.

KYK Yönetim Sistemi için hazırlanmış bir projedir.

Python ile oluşturulmuş döngüler aracılığıyla otomatik olarak

  - Öğrenci yurt başvuruları,
  - İzin talepleri,
  - Ödeme kayıtları
    
düşer. Sistem yönetimi simüle ederek bu taleplerin onaylanması ya da reddedilmesi sağlanabilir, kayıtlara erişilebilir, öğrenci oda değişim işlemleri yapılabilir.

Bu projede Python, FLASK, HTML5, CSS3, JavaScript, MySQL, XAMPP gibi çeşitli yapılar kullanılmıştır.

Flask aracılığıyla Python mimarisinin MySQL tarafına erişimi sağlanmaya çalışılmış,
burada doğan ara katman ihtiyacı XAMPP tarafından sağlananan Apache ve MySQL yapıları ile dinlenerek ihtiyaca göre kullanılmıştır
